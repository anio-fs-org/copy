# @anio-js-foundation/node-fs-copy-sync

Synchronously copy a path.

```js
import nodeFsCopySync from "@anio-js-foundation/node-fs-copy-sync"

console.log(
	nodeFsCopySync("examples/src", "examples/dest")
)
```
